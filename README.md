# bonbonki

A curated channel of Claude Code skills — useful, experimental, and entertaining.

## Install

```
/plugin marketplace add ake/bonbonki
```

Then install individual skills:

```
/plugin install spice-must-flow@bonbonki
```

## Skills

| Skill | Description |
|-------|-------------|
| **spice-must-flow** | Flow state companion — watches your session and provides reflective insights on demand |

## Contributing

Contributions welcome. Skills are accepted via external GitHub refs (your code stays in your repo). Open an issue to discuss before submitting.
